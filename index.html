Embarking on a Web Development Journey: My First Steps

- Introduction

In this blog, I want to share my first experience entering the world of web development. I will narrate the feelings, challenges, and initial achievements I went through. Hopefully, this story can provide inspiration and assistance for those who are also embarking on a similar journey.

- The Beginning of Everything

Initially, entering the world of web development felt like stepping into a wild forest. I was confronted with various programming languages, foreign concepts, and new tools I had never encountered before. It felt like I was on a continent I had never visited before. At first, I felt insecure because almost all of my classmates had a background related to programming. However, the advantage of having such an environment was that it fueled my motivation to learn even harder. Also, if I encountered difficulties in completing my tasks, I could always ask my classmates for help.

- Mastering Programming Languages

After overcoming the feeling of being overwhelmed, I decided to focus on one programming language first. I chose HTML and CSS because I wanted to understand how web design works. To be honest, I enjoyed learning about User Interface, but as time went on, it seemed like I needed a deeper understanding of web development. I started from the basics and completed various online courses while also reading reference books. As for online resources regarding HTML and CSS, I would like to recommend W3schools and FreeCodeCamp.

- Understanding Basic Concepts

I realized the importance of understanding the basic concepts in web development. Mastering HTML for structure, CSS for layout and style, and JavaScript for interactivity are crucial initial steps.

HTML (HyperText Markup Language) is the standard markup language for creating the structure and content of web pages. In HTML, you use tags to mark elements like headings, paragraphs, images, links, forms, and much more.
Example of HTML Usage:

  • !DOCTYPE html: Defines the document type as HTML.
  • html: Starts and ends the HTML element.
  • head: Contains information about the document like title, meta tags, and links to external files.
  • title: Specifies the title of the page displayed in the browser tab.
  • body: Contains the main content of the web page.
  • h1: Heading level 1 (main heading).
  • p: Paragraph.
  • img: Image.
  • a: Link.

CSS (Cascading Style Sheets), CSS is used to control the layout and style of web pages. It allows you to separate structure and design, so you can change the appearance of a web page without altering its structure.

JavaScript, JavaScript is a programming language used to create interactivity on web pages. It enables you to modify content, respond to user actions, and communicate with the server.
