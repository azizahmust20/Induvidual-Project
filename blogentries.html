Embarking on a Web Development Journey: My First Steps

Pendahuluan

Di blog ini, saya ingin membagikan pengalaman pertama saya dalam memasuki dunia pengembangan web. Saya akan menceritakan perasaan, tantangan, dan pencapaian awal yang saya alami. Semoga cerita ini dapat memberi inspirasi dan bantuan bagi mereka yang juga sedang memulai perjalanan serupa.

Awal dari Segalanya

Pada awalnya, memasuki dunia pengembangan web terasa seperti memasuki hutan belantara. Saya berhadapan dengan berbagai bahasa pemrograman, konsep-konsep asing, dan alat-alat baru yang belum pernah saya temui sebelumnya. Rasanya seperti saya berada di benua yang belum pernah saya kunjungi sebelumnya. awalnya, saya sempat merasa insecure karena hampir semua teman kelas saya mempunyai background yang terkait dengan pemrograman. Namun, keuntungan dari mempunyai lingkungan seperti itu adalah dapat memacu semangat saya untuk bisa belajar lebih semangat lagi dan juga jika saya mengalamai kesulitan dalam mengerjakan tugas saya bisa menanyakannya kepada teman-teman kelas saya.

Menguasai Bahasa Pemrograman

Setelah mengatasi perasaan kewalahan, saya memutuskan untuk memfokuskan diri pada satu bahasa pemrograman terlebih dahulu. Saya memilih html dan css, karena saya ingin mengetahui tentang cara design web itu sendiri. sejujurnya, saya senang mempelajari User Interface namun seiring berjalanannya waktu sepertinya saya memerlukan ilmu mendalam mengenai web development. Saya mulai dari dasar-dasar dan menyelesaikan berbagai kursus online serta membaca buku referensi. Adapun sumber website online yang saya ingin rekomendasikan mengenai html dan css adalah W3schools dan FreeCodeCamp.

Memahami Konsep Dasar

Saya menyadari bahwa penting untuk memahami konsep dasar dalam pengembangan web. Menguasai HTML untuk struktur, CSS untuk tata letak dan tata gaya, serta JavaScript untuk interaktivitas adalah langkah awal yang penting.

HTML (HyperText Markup Language), adalah bahasa markah standar untuk membuat struktur dan konten halaman web. Di dalam HTML, Anda menggunakan tag-tag untuk menandai elemen-elemen seperti judul, paragraf, gambar, tautan, formulir, dan banyak lagi.
Contoh penggunaan HTML:

  • !DOCTYPE html: Mendefinisikan tipe dokumen sebagai HTML.
  • html: Memulai dan mengakhiri elemen HTML.
  • head: Berisi informasi tentang dokumen seperti judul, meta tags, dan link ke file eksternal.
  • title: Menentukan judul halaman yang akan ditampilkan di tab browser.
  • body: Berisi konten utama halaman web.
  • h1: Heading level 1 (judul utama).
  • p: Paragraf.
  • img: Gambar.
  • a: Tautan (link).

CSS (Cascading Style Sheets), CSS digunakan untuk mengontrol tata letak dan tata gaya halaman web. Ini memungkinkan Anda untuk memisahkan struktur dan desain sehingga Anda dapat mengubah tampilan halaman web tanpa mengubah strukturnya.

javaScript, JavaScript adalah bahasa pemrograman yang digunakan untuk membuat interaktivitas pada halaman web. Ini memungkinkan Anda untuk mengubah konten, menanggapi tindakan pengguna, dan berkomunikasi dengan server.
